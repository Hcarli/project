Wellcome to Economic Growth Home Page!

In these days, globalization is a fact: nations needs to each other to grow up and development themselves. But do you know how a country grows?

In this web page, I'll try to show how a country grow and develop, and which are the indicators that we need to observer to understand better about growth of nations. To do this, we will analize and data called penn world table, that has many indicators usually used to mesure growth of a country.

For implement this we use variables we get in penn world table at Pennn World Table, download the data in csv file, and after do some analysis about which variables will be used, we transform this csv in a database called growth.db to store the data that will be use.

We treated this data, using pandas library and after it we used to mesure the growth of an nation adopting the Sollow Model
